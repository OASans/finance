# 定期调整

## stock_least_position

* 语法：
  
  * stock_least_position(stock_code: str) -> int
  
* 输入：
  
  * stock_code: 股票代码。
  
* 输出：
  
  * int, 用户设定的最少持仓股票数。
  
* 说明：

  - 返回用户某支股票的最少持仓数。

  - 或许需要访问系统其它地方的数据库（用户指定的最少持仓）。

## stock_volatility

- 语法：
  - stock_volatility(stock_code: str, time: int) -> float
-  输入：
  - stock_code: 股票代码。
  - time: 波动率的时间周期，可以是1, 7, 30，代表当日，近一周，近一月。
- 输出：
  - float, 股票在`time`周期下的波动率。
- 说明：
  - 返回该股票在指定时间周期下的波动率。
  - 或许需要访问系统其它地方的功能接口（波动率计算）。

## stock_macd

- 语法：
  - stock_macd(stock_code: str) -> float
- 输入：
  - stock_code: 股票代码。
- 输出：
  - float, 股票的macd指数。
- 说明：
  - 根据最近26日的收盘价，计算出指数平滑移动平均线（Moving Average Convergence and Divergence，简称MACD）。
  - 或许需要访问外部网站的数据（近期26日收盘价）。

## stock_rsi

- 语法：
  - stock_rsi(stock_code: str) -> dict
- 输入：
  - stock_code: 股票代码。
- 输出：
  - dict, 包含三个key：'rsi6', 'rsi12', 'rsi24'，对应的value均为float。分别为股票的6日、12日、24日相对强弱指数RSI。
- 说明：
  - 根据最近24日的收盘价，计算6日、12日、24日的相对强弱指数RSI
  - 或许需要访问外部网站的数据（近期24日收盘价）。

## stock_kdj

- 语法：
  - stock_rsi(stock_code: str) -> dict
- 输入：
  - stock_code: 股票代码。
- 输出：
  - dict, 包含三个key：'K', 'D', 'J'，对应的value均为float。三个值分别为随机指标中的K, D, J。
- 说明：
  - 根据最近9日的收盘价，计算随机指标中的K, D, J值。
  - 或许需要访问外部网站的数据（近期9日收盘价）。

## stock_roc

- 语法：
  - stock_rsi(stock_code: str, N: int) -> float
- 输入：
  - stock_code: 股票代码。
  - N: 指定与多少天前的收盘价比较。
- 输出：
  - rsi: float, 变动率指标ROC。
- 说明：
  - 根据指定的N，比较本日和N日前的收盘价，返回变动率。
  - 或许需要访问外部网站的数据（近期N日收盘价）。

## stock_sharpe

- 语法：

- 输入：

- 输出：
  - rsi: float, 夏普比的数值。
- 说明：
  - 以日收益率为标准，返回股票的夏普比。
  - 或许需要访问外部网站的数据（日收益率）。
